# Pangeran For Web
#### *Membuat virtual host di apache server dengan mudah dan cepat di VPS.*

#### *Cara menjalankan*
- git clone https://github.com/Xcod3bughunt3r/PangeranWeb.git
- cd PangeranWeb/
- chmod -R 777 *
- sudo ./pangeranweb.sh
